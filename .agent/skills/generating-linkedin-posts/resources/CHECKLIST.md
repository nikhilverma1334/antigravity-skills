# LinkedIn Content Checklist

- [ ] **First 2 lines**: Do they create curiosity?
- [ ] **White Space**: Are there no more than 3 lines of text per paragraph?
- [ ] **Formatting**: Are bullets or emojis used correctly for readability?
- [ ] **CTA**: Is there a single, clear question at the end?
- [ ] **Research**: Does the post contain at least one specific fact, stat, or unique insight?
- [ ] **Hashtags**: Are there 3-5 tags at the bottom?
- [ ] **Images (Recommended)**: Suggest an image or chart type for the user to include.
